C++ library to handle converting encoder angle readings to odom values

\doxysection*{Building}

This library is built with cmake, to build run the following


\begin{DoxyCode}{0}
\DoxyCodeLine{git clone https://github.com/lucimobility/encoder-\/to-\/odom-\/library.git}
\DoxyCodeLine{cd encoder-\/to-\/odom-\/library}
\DoxyCodeLine{mkdir build}
\DoxyCodeLine{cd build}
\DoxyCodeLine{cmake ..}
\DoxyCodeLine{make -\/jn (n is number of parallel jobs you want a good value is usually 4)}

\end{DoxyCode}
